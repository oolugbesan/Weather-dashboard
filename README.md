# Weather Dashboard 🌥️ 

A simple weather dashboard that fetches real-time weather data using the OpenWeather API.

## Features
- Search for a city
- View current temperature
- View weather condition
- View humidity
- View wind speed

## Technologies Used
- HTML
- CSS
- JavaScript
- OpenWeather API
- JSON

## What I Learned
- API integration
- Async/await
- JSON parsing
- DOM manipulation
- Error handling
